# Build Tools & Asset Pipeline

Utilities for building, testing, and deploying game projects.

## Build Scripts

- `build.sh` — Universal build script for all platforms
- `clean.sh` — Clean build artifacts
- `package.sh` — Package for distribution

## Asset Tools

- `texture_optimizer/` — Optimize textures for different platforms
- `audio_converter/` — Convert audio formats
- `mesh_compiler/` — Compile 3D models
- `sprite_sheet_generator/` — Generate sprite sheets from assets

## Testing & Validation

- `test_runner.sh` — Run automated tests
- `performance_profiler/` — Profile performance
- `asset_validator/` — Validate asset compliance

## CI/CD Pipeline

- GitHub Actions workflows
- Automated testing on commits
- Build artifact generation
- Deploy scripts

See `.github/workflows/` for configuration.
