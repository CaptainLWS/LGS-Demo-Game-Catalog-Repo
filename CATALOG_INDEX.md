## Key Directories

- `/apps/` — Mobile game projects
- `/console/` — Console and handheld game projects
- `/computer/` — Desktop/PC game projects
- `/web/` — Browser-based game projects
- `/systems/` — Shared engines, physics, networking, asset management
- `/tools/` — Build scripts, asset pipelines, debugging tools
- `/docs/` — Architecture, API docs, design documents
- `/qa/` — QA pipeline, test cases, bug tracking

## Getting Started

1. **Choose a platform** — Mobile, Console, Computer, or Web
2. **Pick a project** — Click through to the platform-specific catalog
3. **Read the project README** — Each project has detailed setup instructions
4. **Install dependencies** — Follow build prerequisites
5. **Run the build** — Use the provided build scripts
